# SEO Report Templates

Reference file with report templates for different audiences. Used by the seo-reporting skill.

## Executive Report Template

One-page summary for C-suite and leadership. No jargon, business impact focus.

### Structure

**Headline:** One sentence summarizing the period's performance.

**KPI Dashboard:**

| KPI | Current | vs Last Period | vs Target |
|-----|---------|---------------|-----------|
| Organic revenue | $X | +/-% | On/Off track |
| Organic leads | X | +/-% | On/Off track |
| Organic traffic | X sessions | +/-% | On/Off track |

**What happened:** 2-3 sentences explaining the numbers in business terms.

**Top wins:** Bullet list of 2-3 achievements with business impact.

**Risks:** Any concerns that leadership should be aware of.

**Next steps:** 2-3 planned actions for the next period.

### Example narrative
> Organic search revenue grew 12% quarter-over-quarter, driven by new product pages targeting high-intent keywords. We're on track to hit the annual organic traffic target. Next quarter, we'll focus on expanding into the [topic] category, which represents a $X opportunity based on search demand.

## Client Report Template

Moderate detail for marketing managers and stakeholders. Some SEO terms OK, but explain technical concepts.

### Structure

**1. Executive Summary** (half page)
- Period headline
- KPI summary table (5-7 metrics)
- Key takeaway

**2. Organic Performance** (1 page)
- Traffic trends with visualization recommendations
- Organic share of total traffic
- Year-over-year comparison
- Notable changes and explanations

**3. Search Visibility** (1 page)
- Keyword rankings for target keywords (table: keyword, current position, previous, change)
- New keyword rankings gained
- SERP feature wins/losses

**4. Content Performance** (1 page)
- Top landing pages by organic traffic
- New content published and early performance
- Content optimization results

**5. Technical Health** (half page)
- High-level site health score
- Critical issues fixed
- Outstanding issues

**6. Competitive Landscape** (half page)
- Visibility trend vs key competitors
- Notable competitive changes

**7. Actions & Next Steps** (1 page)
- Completed this period (with results)
- Planned for next period (with expected impact)
- Blocked items (with what's needed)

**8. Appendix**
- Detailed data tables
- Methodology notes
- Glossary of SEO terms used

## Technical Report Template

Full detail for development teams and technical SEO implementers. Technical SEO terms expected.

### Structure

**1. Summary** (brief)
- Critical issues requiring immediate action
- Issue counts by severity

**2. Findings** (detailed, by category)

For each finding:

```
### [Finding Title]

**Severity:** Critical / High / Medium / Low
**Category:** Crawlability / Indexation / Speed / Structured Data / Mobile / Security
**Affected URLs:** [count] pages ([sample URLs])

**Issue:** [Technical description of the problem]

**Impact:** [How this affects rankings/indexing/user experience]

**Fix:**
1. [Step-by-step implementation instructions]
2. [Code examples if applicable]
3. [Verification steps]

**Priority:** [Immediate / This sprint / Backlog]
```

**3. Implementation Checklist**
- [ ] Fix 1 — [description] — [assignee] — [deadline]
- [ ] Fix 2 — [description] — [assignee] — [deadline]

**4. Verification Plan**
- How to verify each fix was implemented correctly
- Expected metrics improvement post-fix
